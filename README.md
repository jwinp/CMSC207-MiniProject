## **Project Overview**

WINstagram is a basic personal web profile mini-project built as a single-page website using HTML, CSS and JavScript. It showcases:

- Navigation menu with placeholder links.
- About Me section with a profile image and dynamic fun fact button.
- Hobbies & Interests gallery with interactive images.
- Contact information with social media links (placeholders).
- Responsive design for desktop and mobile screens.
- Basic interactivity using JavaScript (image swaps, like/share/save buttons).

This project demonstrates the use of HTML, CSS, and JavaScript to create a visually appealing, interactive, and mobile-friendly webpage.
